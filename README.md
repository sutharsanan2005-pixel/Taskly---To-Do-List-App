# Taskly — To-Do List App

A clean, dark-themed to-do list app built with plain HTML, CSS, and JavaScript.

## Features
- Add and delete tasks
- Mark tasks as complete
- Filter by All / Active / Done
- Tasks saved with localStorage
- Fully responsive

## How to Run
Just open index.html in any browser!

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
